buildRouteMap()[source]¶

Build the URL route map for all routes in the Flask app.

Returns in self.results a key ‘urlmap’ of dictionary of routes. Syntax: {blueprint: {endpoint: {‘methods’:x, ‘url’:x} } E.g. getSpectrum method urlmap = {‘api’: {‘getspectra’: {‘methods’:[‘GET’,’POST’], ‘url’: ‘/api/cubes/{name}/spectra/{path}’} } }

urls can now easily handle variable replacement in real code; MUST use keyword substitution E.g. print urlmap[‘api’][‘getspectra’][‘url’].format(name=‘1-209232’,path=’x=10/y=5’) returns ‘/api/cubes/1-209232/spectra/x=10/y=5’
